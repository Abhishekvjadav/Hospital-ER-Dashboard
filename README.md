## 📌 Project Overview

This project presents a comprehensive **Hospital Emergency Room Dashboard** developed using **Power BI** to analyze patient visits, operational efficiency, and healthcare performance over a 19-month period.

The dashboard enables hospital administrators and decision-makers to monitor patient flow, identify bottlenecks, optimize staffing, and improve patient care through interactive visualizations and key performance indicators (KPIs).

---

## 🎯 Business Problem

Emergency departments generate large volumes of patient data daily. Without proper visualization, it becomes difficult to identify:

- Peak patient hours
- Long waiting times
- Referral trends
- Admission patterns
- Patient demographics
- Operational inefficiencies

This dashboard transforms raw hospital data into actionable business insights to support data-driven decision making.

---

## 🎯 Objectives

- Monitor emergency room performance
- Analyze patient wait times
- Track patient satisfaction
- Understand referral patterns
- Identify peak operating hours
- Improve resource allocation
- Support healthcare decision-making
